OC Project 6

Github

Overview

The sixth project in the OpenClassrooms program. I was given some design documents, and expected to create a social media like site that was focused on allowing users to post hot sauce reviews. I worked on both the front and back end for this project.

Skills

This project gave me more practice with the back end server, but also allowed to learn how to write the front end code that would interface with it. I learned how to make calls to the API and handle error cases that may arise.

Thanks for checking out the site!
