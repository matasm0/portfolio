Overview

The fifth project in the OpenClassrooms program. I was given a completed front end of an example online store, and was tasked with creating the backend server that the site would communicate with.

Skills

I learned how to write a server using Express, and set up the APIs in such a way that the front end site can communicate with it. Through that, I used APIs as well as routes and controllers.

Thanks for checking out the site!
